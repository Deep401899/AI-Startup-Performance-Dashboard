# AI-Startup-Performance-Dashboard

## Overview

This project analyzes the performance of an AI SaaS startup using Power BI.

The dashboard helps management monitor:

- Revenue
- Customer Growth
- Churn
- Marketing Performance
- Subscription Plans
- Customer Satisfaction

---

## Tools Used

- Power BI
- DAX
- Power Query
- Excel / CSV

---

## Dataset

1000 AI startup customers

Features include:

- Customer ID
- Region
- Industry
- Subscription Plan
- Revenue
- Marketing Channel
- Churn Risk
- Customer Satisfaction

## Key Insights

- Enterprise plan generates the highest revenue.
- Google Ads brings the most customers.
- Technology industry contributes the largest revenue.
- Customer satisfaction remains above 4.2.
